# Gunners-Gear
A Full Stack MERN application to browse Arsenal related merchandise and buy them.

https://gunners-gear.herokuapp.com

## Features
1 - User can create an account to log in or login as a guest.

2 - User can browse Arsenal merchandise and add product(s) to the cart.

3 - User can increase/decrease quantity of any item in the cart or remove it completely.

4 - User can process an order and view their order history in the accounts page.


## Technologies Used
1 - React (react hooks, react router)

2 - Node.js

3 - Express

4 - MongoDB

5 - Mongoose

6 - Joi

7 - bcrypt

8 - json web token

9 - CSS

10 - Material UI
